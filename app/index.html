Privacy & Security

This project aims to secure your website and keep adversaries crawlsers and bots away from your personal website. The typical and ineffective way of doing this is collecting a huge list of malicious keywords which can be used to identify unwanted user agents. Mainting this list is a hassle. I say there's a better way!

Approaching the Problem

Instead of finding the advesaries, why not find trusted browsers instead. Using the sliders below you can create your own personal profile that matches only the browsers, operating systems, devices & brands you trust! The end result is a powerfull and personalized regex that can be used in your web server configuration like nginx and apache to keep bad guys out
